# Thanos

## Linux

### ThanosLocal

1. 切换到要执行的的目录

2. ```bash
   path/ThanosLocal.sh	#path为sh所在目录
   ```



### ThanosRoot

​	直接执行

```bash
path/ThanosRoot.sh
```

ThanosRoot.sh 删除一半的文件
